A Galaga like arcade style game. Created using C++, OpenGL, and libraries like the sound engine irrKlang and freetype. 

I have an itch.io page for this game with screenshots and a download as well! Please take a look: 
Itch.io: https://justin-s-gonzales.itch.io/war-space

All textures were created by me. Sounds were sourced from freesound.org.

There is a video demonstration included in the files or you can click here to view it on LinkedIn:
